# Real-Time Collaborative Workspace

A modern Agile Management Tool that enables teams to collaborate, manage projects, track tasks, and communicate in real time. This platform helps teams stay organized, improve productivity, and streamline project workflows.

## 🚀 Features

### User Management

* User Registration and Login
* Secure Authentication
* Role-Based Access Control
* User Profiles

### Workspace Management

* Create and Manage Workspaces
* Invite Team Members
* Manage Workspace Settings
* Real-Time Collaboration

### Project Management

* Create and Manage Projects
* Project Dashboard
* Project Progress Tracking
* Team Assignment

### Task Management

* Create, Update, and Delete Tasks
* Assign Tasks to Team Members
* Set Priorities and Deadlines
* Task Status Tracking
* Kanban Board Support

### Real-Time Collaboration

* Instant Updates Across Team Members
* Real-Time Notifications
* Activity Tracking
* Collaborative Workspace Environment

### Dashboard

* Project Overview
* Task Statistics
* Team Performance Metrics
* Recent Activities
* Workspace Analytics

## ⚙️ Installation

### Clone Repository

```bash
git clone <repository-url>
cd Agile-management-tool
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

### Backend Setup

```bash
cd backend
npm install
npm start
```

## 🔐 Environment Variables

Create a `.env` file in the backend directory:

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```
